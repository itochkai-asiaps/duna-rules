## Что сделано

### 1. Сбор правил
- Выгружены **56 страниц** с сайта duna2026.ru (Tilda) — текст + PDF
- Выгружены **2 Google Docs**: Медицина (54K), География (87K)
- Выгружены **3 Google Sheets**: Приложения 1-3 к закону о Фафрелахе (Статусы, Обязанности, Права)
- Приложения 4-8 — внутри текста закона на сайте, не требуют отдельной выгрузки
- **Итого:** 58 текстовых документов + 3 CSV-таблицы, ~800K символов

### 2. PDF-коллекция (`rules/`)
- `01-общие-правила\` — 11 PDF
- `02-модельные-правила\` — 26 PDF (вкл. полную Медицину из Google Docs)
- `03-материалы\` — 10 PDF (вкл. полную Географию из Google Docs)
- `04-законы-империи\` — 9 PDF + 3 xlsx + 3 csv (приложения Фафрелаха)
- `05-рубежи-саммари\` — 1 PDF (памятка для команды)

### 3. PWA-справочник (`app/`, `index.html`)
- Одна HTML-страница, все правила вшиты (1.1 МБ)
- Полнотекстовый поиск по всем 58 документам
- Меню-бургер с навигацией по разделам
- Таблицы Фафрелаха — HTML-рендеринг
- Офлайн через Service Worker (на HTTPS)
- Опубликован на GitHub Pages ✅

### 4. Скрипты обновления (`scraper/`)
- `update-rules.mjs` — пересобрать весь корпус (сайт + Google Docs)
- `config.json` — список всех источников (56 URL + 2 Google Doc ID)
- Для обновления перед игрой: `node update-rules.mjs` → пересобрать index.html → `git push`

## Проблемы / Баги

### iOS
- **Локальный HTML не работает** — iOS не исполняет JS из локальных файлов
- Koder зависает на большом файле
- **Решение:** GitHub Pages (работает ✅)
- Quick Look не показывает иконку Safari — нужно через «Поделиться» → «Ещё» включить

### PWA
- `escRegex` был сломан (неправильный regex character class) — **исправлено**
- `behavior:'smooth'` глючит на Safari — **убрано**
- Service Worker вшит инлайн, manifest — data URI
