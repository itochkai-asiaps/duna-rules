# Дюна 2026 — Офлайн-справочник правил

## Что это

Полностью автономное PWA-приложение со всеми правилами ролевой игры «Дюна 2026». Работает **без интернета** после первой загрузки.

---

## Установка на iPhone (iOS)

> ⚠️ Встроенный предпросмотр «Файлов» (Quick Look) **не исполняет JavaScript** — поиск и меню работать не будут. Нужно открыть через приложение с поддержкой JS.

### Способ 1 — Koder (рекомендуется, бесплатно)

1. Установи **Koder** из App Store (бесплатный редактор кода)
2. Доставь `index.html` на iPhone (AirDrop / Telegram / iCloud)
3. В **Файлах** → долгий тап на `index.html` → **Поделиться** → **Koder**
4. В Koder нажми кнопку **▶ Play** внизу экрана
5. Откроется полноценный браузер — поиск, меню, таблицы работают

### Способ 2 — Safari (если получится включить)

1. **Файлы** → долгий тап → **Поделиться** → проведи влево → **Ещё**
2. Включи **Safari** → **Готово**
3. Снова долгий тап → **Поделиться** → **Safari**

> Safari может отсутствовать в «Ещё» — тогда только Koder.

---

### Как доставить папку на iPhone

**AirDrop (быстрее всего):**
1. На компьютере правый клик на папке `duna-app` → **Поделиться** → **AirDrop** → iPhone
2. На iPhone папка появится в приложении **Файлы**

**Telegram:**
1. На компьютере выдели все файлы из `duna-app` → правый клик → **Сжать в ZIP**
2. Отправь архив себе в **Избранное** Telegram
3. На iPhone скачай архив → тапни по нему → распакуется в **Файлы**

**iCloud Drive:**
1. Скопируй папку `duna-app` в iCloud Drive с компьютера
2. На iPhone открой **Файлы** → iCloud Drive → `duna-app`

## Установка на Android

### Способ 1 — Через Chrome

1. Скопируй папку `duna-app` на телефон (по USB-кабелю в папку Download)
2. Открой **Chrome**
3. В адресной строке введи: `file:///sdcard/Download/duna-app/index.html`
4. Нажми на три точки (меню) → **Добавить на главный экран**
5. Нажми **Добавить**

### Способ 2 — Через Telegram (аналогично iPhone)

1. Отправь ZIP-архив папки себе в Telegram
2. Скачай на телефоне → распакуй
3. Открой `index.html` через Chrome
4. Меню → **Добавить на главный экран**

## Как пользоваться

- **Поиск** — строка вверху. Вводи запрос (например, «ранение», «фафрелах», «пряность»)
- **Результаты** — показывают параграфы с подсветкой найденных слов и названием раздела
- **Разделы** — кнопка-бургер слева открывает список всех документов по папкам
- **Таблицы** — приложения 1/2/3 к закону о Фафрелахе открываются в виде читаемых таблиц

---

## Обновление правил

Если правила на сайте обновились:

1. Запусти `node update-rules.mjs` из папки `duna-scraper` (обновит корпус с сайта)
2. Запусти `node build.mjs` из папки `duna-app` (пересоберёт index.html с новыми текстами)
3. Обнови файл на телефоне (отправь заново)

## Состав папки

| Файл | Назначение |
|------|-----------|
| `index.html` | Приложение: поиск + все правила внутри (1.1 МБ) — **единственный нужный файл** |
| `build.mjs` | Скрипт пересборки при обновлении правил |
| `*.xlsx` / `*.csv` | Приложения к закону о Фафрелахе |
| `sw.js`, `manifest.json` | Для справки (в `index.html` вшиты, отдельно не нужны) |
