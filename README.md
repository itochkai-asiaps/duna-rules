# Дюна 2026 — Офлайн-справочник правил

Полностью автономное PWA-приложение со всеми правилами ролевой игры «Дюна 2026». Работает **без интернета** после первой загрузки.

---

## Установка на iPhone (iOS)

> iOS **не позволяет** открывать локальные HTML-файлы с JavaScript. Все браузеры (Safari, Chrome) используют движок WebKit и не могут исполнить JS из файла на устройстве. Единственный рабочий способ — GitHub Pages.

### Способ 1 — GitHub Pages (рекомендуется)

1. Открой Safari на iPhone
2. Перейди по ссылке: **https://itochkai-asiaps.github.io/duna-rules**
3. Дождись загрузки страницы (нужен интернет только в первый раз)
4. Нажми **Поделиться** → **На экран «Домой»** → **Добавить**

Готово. Иконка «Дюна 2026» на домашнем экране. После первого открытия работает **полностью офлайн** — можно включать авиарежим и проверять поиск.

### Способ 2 — Koder (если GitHub Pages недоступен)

1. Установи **Koder** из App Store
2. Доставь `index.html` на iPhone (AirDrop / Telegram / iCloud)
3. **Файлы** → долгий тап → **Поделиться** → **Koder**
4. Нажми **▶ Play** внизу

> Koder может зависнуть на большом файле (1.1 МБ). GitHub Pages надёжнее.

## Обновление правил

Если правила на сайте обновились:

1. `node update-rules.mjs` — пересобрать корпус с сайта
2. `node build.mjs` — пересобрать index.html
3. `git push` — обновить GitHub Pages
4. На iPhone: открыть Safari, смахнуть страницу вверх (перезагрузить), готово
